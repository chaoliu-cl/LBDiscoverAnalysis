# Calculate ABC score based on specified method

Calculate ABC score based on specified method

## Usage

``` r
calculate_score(a_b_score, b_c_score, method)
```

## Arguments

- a_b_score:

  A-B association score

- b_c_score:

  B-C association score

- method:

  Scoring method: "multiplication", "average", "combined", or "jaccard"

## Value

Calculated score
